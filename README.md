# Dice Mage

## Overview

Dice Mage is a cli application for generate dice-ware compatible pass-phrases via command line.

## Commands

### Generate a pass-phrase

command:  generate  
alias: g  

options:

* Total Words
    * specify the total words to generate for the pass-phrase
    * -t or --total-words
    * default value is 5

example:

```bash

dice-mage generate -t 6

```
